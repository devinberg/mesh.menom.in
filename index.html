Mesh Menomin

A mesh network is made up of an array of direct wireless connections between individual devices that provides for communication among all of the connected devices independent of the internet, wifi, or cellular connections. This can be helpful in situations when communication by normal means are unavailable. Mesh Menomin is a local group interested in broadening participation in mesh networking in the Menomonie Area. There is also a broader, Wisconsin mesh network called Meshconsin that you might be interested in participating in.

What is Meshtastic?

Meshtastic is an "an open source, off-grid, decentralized, mesh network built to run on affordable, low-power devices." The network is designed to operate using the LoRa protocol and within the license-free ISM frequency band at 915 MHz.

How to get started.

  1. Purchase a compatible device. There are many suitable devices that can get you started. We recommend:
    • Heltec V3: An inexpensive device that relies on your phone for programming and messaging, but can receive and repeat messages independently.
    • Lilygo T-deck Plus: A standalone device that includes an integrated GPS and battery. A connected phone is not necessary to operate this device, but it can make programming it easier.
  2. Flash the firmware.
  3. Program your device. Recommended settings:
    • MQTT: Disabled or Ignored
    • Device Role: Client (NOT Router_Client)
    • Hop Count: 3
    • Broadcast Interval (All): Minimum of 1 hour
  4. [optional] Join the MeshMenomin channel for local connections (see sidebar).
  5. Keep you device powered as you move about to participate in the network.
